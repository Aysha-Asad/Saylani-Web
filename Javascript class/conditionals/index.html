Definition Conditionals are used to perform different actions based on different conditions.
In JavaScript, we commonly use if, else if, and else statements to implement conditional logic.
1. if statement: It executes a block of code if a specified condition is true.
2. else if statement: It specifies a new condition to test if the first condition is false.
3. else statement: It executes a block of code if all previous conditions are false.
Example:
let usertype = prompt("Enter user type");
if (usertype === "admin")
{ alert("You are logged in as admin."); }
else if (usertype === "user") {
alert("You are logged in as user."); }
else { alert("You are not logged in."); }
